1. How does inheritance work in CSS?

  1. Styles are inherited only by child elements, so the color is blue.
  2. Styles are inherited only by parent elements, so the color is black (browser default).
  3. Styles are inherited by child elements and can be overridden, so the color blue, but can be changed by the child styling.

Example HTML:

    <div class="parent">
      <p class="child">What is my color?</p>
    </div>

Example CSS:

    .parent {
       color: blue;
       font-weight: 400;
    }
            
    .child {
      font-weight: bold;
    }

2. Which of the following methods is incorrect for defining styles in CSS?

  1. Nesting styles
  2. Inline style
  3. Writing styles in a JavaScript file

Example HTML:

    <p style="color: red;">Text</p>

Example CSS:

    .nested .nested__item {
       font-size: 14px;
    }

Example JS:

    document.getElementById("el").style.color = "red"

3. What is "specificity" of selectors in CSS?

  1. It determines how many times a selector is used in the code.
  2. It determines the order in which style declarations are applied.
  3. It determines the priority of a selector when overriding styles.

Example HTML:

    <p id="myId" class="my-class">What is my color?</p>

Example CSS:

    p {
       color: blue;
    }
            
    .my-class {
        color: red;
    }
            
    #myId {
       color: green;
    }

4. How does the order of styles change in CSS?

  1. From top to bottom.
  2. From bottom to top.
  3. It doesn't matter; the order has no impact.

Example CSS:

    p {
      color: red; 
    }
    
    p {
      color: blue; 
    }

5. What does the "!important" expression mean in CSS?

  1. It instructs the browser to ignore that style declaration.
  2. It gives higher priority to the style than others.
  3. It is an error and should not be used.

Example CSS:

    p {
      color: blue !important;
    }
    
    p {
      color: red;
    }

6. What are the principles of KISS and DRY in CSS coding?

  1. KISS - "Keep It Simple, Stupid"; DRY - "Don't Repeat Yourself."
  2. KISS - "Keep It Short and Sweet"; DRY - "Do Rewrite Yourself."
  3. KISS - "Keep It Stylish and Simple"; DRY - "Do Repeat Yourself."
